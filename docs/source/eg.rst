===========
 Exemple 
===========

Pour exécuter le programme dans votre environnement Python par défaut,
il suffit d'entrer, en ligne de commande:

.. code-block:: console

    $ python3 base.py

sur toute variante Linux/UNIX/BSD/MacOS, et la commande équivalente

.. code-block:: console

    $ py -3 base.py

sur Windows. Vous pouvez aussi exécuter votre programme directement dans 
votre IDE pendant le développement, si vous utilisez par exemple:

- `Visual Studio`_
- :external+python:module:idle
- TextMate_
- Spyder_
- PyCharm_

et plusieurs autres. Vous pouvez vous référer à la page de la `documentation
officielle de Python sur l'utilisation et configuration`_.

.. _documentation officielle de Python sur l'utilisation et configuration:
    https://docs.python.org/3/using/index.html

.. _Visual Studio:
    https://visualstudio.microsoft.com

.. _IDLE:
    https://docs.python.org/3/library/idle.html

.. _TextMate:
    https://macromates.com/

.. _Spyder:
    https://www.spyder-ide.org/

.. _PyCharm:
    https://www.jetbrains.com/pycharm/
